# Andela_Nikhil_MCP

A demonstration project for agent lifecycle logging and MCP server management.

## Structure
- `nikhil_agent_logger.py`: Custom agent lifecycle logging hooks.
- `nikhil_mcp_manager.py`: Manages multiple MCP servers.
- `nikhil_demo.ipynb`: Example notebook for running the agent and MCP servers.
- `sandbox/`: Output directory for agent results.

## Requirements
- Python 3.8+
- `agents` package (and dependencies)
- `python-dotenv`

## Usage
1. Install dependencies:
   ```sh
   pip install -r requirements.txt
   ```
2. Set up your `.env` file with any required API keys (e.g., `BRAVE_API_KEY`).
3. Run the notebook `nikhil_demo.ipynb` in Jupyter or VS Code.

---
This project is a refactored version of a lifecycle hooks demo, with all names and files changed for clarity and uniqueness.
